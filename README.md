# 2B OCR 识别系统

## 项目简介

本项目是计算机视觉实践题 **2B OCR 文字识别系统** 的完整实现，目标是从图片中识别中文、英文、数字和简单表单文字，并输出识别文本、平均置信度和预处理结果图。

项目同时提供：

- Python OCR 验证版本
- C++ 命令行批量识别版本
- Windows 原生 Qt 图形界面版本
- 26 张测试图片
- 预处理输出图、CSV 结果表格、运行日志和项目报告

## 功能特点

- 支持单张图片 OCR 识别
- 支持文件夹批量 OCR 测试
- 支持中文、英文和数字识别
- 使用 OpenCV 进行图像预处理
- 使用 Tesseract OCR 进行文字识别
- 输出平均置信度
- 保存预处理后的图片
- 导出 UTF-8 CSV 结果表格
- 提供中文 Qt 图形界面

## 运行方式

### 方式一：启动 Qt 界面

在 Windows 中双击：

```text
启动2B_Qt界面.bat
```

界面中可以打开图片、查看原图或预处理图、执行 OCR 识别，并保存识别文本。

### 方式二：命令行批量测试

在项目目录中运行：

```text
运行2B_CLI测试.bat
```

或手动运行：

```bat
ocr_cli.exe 输入图片或文件夹 输出文件夹 CSV结果路径
```

示例：

```bat
ocr_cli.exe 01_测试原图 02_预处理输出图 03_结果表格/2B完整OCR识别结果_26张.csv
```

## 预处理说明

OCR 识别前需要先改善图片质量。本项目使用的预处理包括：

| 步骤 | 作用 |
|---|---|
| 灰度化 | 去掉颜色干扰 |
| 放大 | 提高小字识别效果 |
| 中值滤波 | 减少噪声 |
| 二值化 | 增强文字与背景的对比 |

## 局限性

- Tesseract 对复杂中文场景、表格线、旋转文字和低质量图片仍可能出现误识别。
- 平均置信度只能作为参考，不代表每个字都识别正确。
- 表格、票据类图片如果版式复杂，可能需要更强的版面分析方法。
